Descripción para iniciar el entorno venv (y ejecutar bot):

1- Instalar librerías necesarias:
pip install python-telegram-bot
pip install google-generativeai

2- En Windows Terminal o CMD: cd ruta\al\proyecto

3- En Windows Terminal o CMD: python -m venv venv (iniciar entorno)

4- En Windows Terminal o CMD: venv\Scripts\activate (activar entorno)

5- En Windows Terminal o CMD: python bot_sismos.py
